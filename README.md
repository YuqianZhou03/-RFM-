# -RFM-
本项目基于 100 万条淘宝用户行为数据，构建了从全链路诊断到精细化运营的商业分析闭环。  首先，利用漏斗模型定位业务瓶颈，发现点击至意向环节跳失率高达 90.67%，而意向转化率达 24.44%。针对流失问题，通过 RFM 模型对用户进行分层，精准锁定活跃度下降但具高潜力的“重要唤回客户”。  为科学验证策略，项目设计了 A/B Testing 模拟实验，对召回组发放优惠券。统计学检验显示，实验组转化率从 2% 提升至 3%，相对提升 50%；P 值 (0.0017) 远小于 0.05，证实了营销干预的显著性。  项目完整展示了数据清洗、R/F 指标计算、卡方检验 及可视化分析，体现了运用数据驱动业务增长的实战能力。
