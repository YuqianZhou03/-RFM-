Taobao User Behavior Analysis & Growth Hacking
关键词：用户路径分析 | RFM 精细化建模 | A/B Testing 策略验证

本项目深度挖掘了 100 万条淘宝用户行为日志，实现了一套从流量诊断到策略回测的完整商业分析方案。

核心工作：
转化漏斗诊断：通过多跳转化建模，精准定位流量流失点。数据显示，PV 到加购/收藏环节的跳失率高达 90.67%，是提升全链路 GMV 的关键切入点。

RFM 用户画像：基于 Recency 和 Frequency 维度构建动态分层模型。重点锁定了处于流失边缘但具备高忠诚度潜力的“重要唤回客户”，为差异化运营提供数据支撑。

策略实验验证 (A/B Test)：针对召回客群模拟发放优惠券策略。通过卡方检验进行显著性验证，结果显示：实验组转化率从 2% 提升至 3%，实现 50% 的相对增长，且 P-value (0.0017) 证实该提升具备极强的统计学显著性。

技术栈：
Pandas, NumPy, Scipy (Stats), Matplotlib, Seaborn
